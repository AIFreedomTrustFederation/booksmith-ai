# Federation Library Cross-References

This file maps how books in the AI Freedom Trust Federation Library should reference one another.

## Foundational Flow

```text
Universal Process Theosophy
→ Towards a Unified Reality
→ Coincidence, Synchronicity, and the Eternal Now
→ Greenshift Theory
→ Federation Theory
→ DynastyLink / AI Stewardship / Booksmith AI Manual
```

## Core Reference Relationships

- `universal-process-theosophy` is the metaphysical root for the whole library.
- `towards-a-unified-reality` is the physics-to-metaphysics bridge.
- `coincidence-and-the-eternal-now` develops meaning, synchronicity, and observer-frame theory.
- `krystal-spiral` develops recursive consciousness and harmonic symbolic geometry.
- `greenshift-theory` develops regenerative coherence, death/resurrection symbolism, and toroidal center-zone dynamics.
- `the-green-zone` translates coherence and regeneration into emotional sobriety, recovery, and lived practice.
- `federation-theory` translates coherence into governance, sovereignty, subsidiarity, and collective intelligence.
- `dynastylink` applies federation theory to trust identity and intergenerational legacy.
- `aift-constitution-and-doctrine` turns federation theory into operational doctrine.
- `ai-stewardship-and-alignment` applies the doctrine to AI systems, local-first architecture, and human-AI cooperation.
- `booksmith-ai-manual` explains how the library is authored, edited, cited, packaged, and released.
- `civilizational-resilience` applies the framework to extinction risk, adaptation, and long-term continuity.
- `library-of-the-federation` documents the archive, canon, knowledge graph, and preservation strategy.

## Citation Practice

Books should cite primary sources, prior federation books, source documents, and relevant repository files. Internal cross-references should use stable book slugs and chapter identifiers wherever possible.
