# Federation Library Canon

The Federation Library Canon defines the shared doctrinal and publishing backbone for AI Freedom Trust Federation books.

## Canon Principle

Each book is sovereign enough to stand alone and federated enough to support the whole library.

## Canonical Commitments

1. **Author sovereignty:** human judgment, authorship, approval, and consent remain central.
2. **Local-first publishing:** core creation, revision, typesetting, and export workflows should remain self-hostable.
3. **Open-source readiness:** default tools, fonts, templates, and models should prefer open and commercially compatible licenses.
4. **Inspectable knowledge:** manuscripts, citations, prompts, glossary terms, build logs, and provenance should remain visible and reproducible.
5. **Federated coherence:** each book should reference related books, shared terms, and source lineages without collapsing into one giant undifferentiated work.
6. **Doctrine with humility:** speculative metaphysical claims should be clearly distinguished from established scientific claims, while still allowing disciplined synthesis.
7. **Evolutionary continuity:** the library should support individual creativity, collective intelligence, intergenerational learning, and reduced extinction risk.

## Canonical Library Formula

```text
Federated Library = Sovereign Books + Shared Canon + Cross References + Provenance
```
