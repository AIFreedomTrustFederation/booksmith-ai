# Font Assets

Place project-specific open fonts here when a book needs bundled typography.

Default Booksmith templates should prefer installed OFL/open fonts such as Noto,
EB Garamond, Cormorant Garamond, Alegreya, Libertinus, and TeX Gyre. Custom font
files can be loaded with `fontspec` using a `Path=assets/fonts/` option.

Do not commit proprietary commercial fonts unless the project license explicitly
allows redistribution.
