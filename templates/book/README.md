# Book Template

Use this template for every new federated book project.

## Required Files

```text
book.config.json
manuscript/00-preface.md
manuscript/references.md
sources/
assets/
latex/
exports/
```

## Authoring Rule

The template should help authors preserve creative control while keeping outputs reproducible, citable, and ready for AI Freedom Trust Publishing review.

## Metadata Rule

Every book must declare:

- title
- slug
- series
- status
- purpose
- audience
- canonical terms
- related books
- source lineage
- publishing packet targets
