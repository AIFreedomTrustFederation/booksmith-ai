# CT011 — Coincidence Network

## Setup

Imagine all events as nodes in a graph.

At first, the observer sees only two connected events:

```text
A -- B
```

Later, the observer discovers a wider network:

```text
A -- B -- C -- D -- Omega
```

## Question

Was the relationship created, or merely discovered?

## FHQCM Interpretation

Coincidence is observer recognition of relationship within a wider field.

## Alternative Explanations

- statistical clustering
- apophenia
- confirmation bias
- narrative reconstruction

## Falsifier

If perceived relationships cannot be distinguished from ordinary statistical and cognitive effects, the claim must be weakened.
