# FHQCM Equation Canon

This file collects equations and mathematical symbols used or planned in the FHQCM manuscript. It is a notation registry for editorial cleanup, not a claim that every equation proves the model.

## Editorial Rules

- Keep established equations separate from symbolic or metaphysical extensions.
- Convert extracted text equations into valid LaTeX.
- Do not imply that symbolic equations have the same empirical status as established physics.
- Every equation used in the final book should include a plain-language explanation.

## Established / Standard Equations To Verify

### Mandelbrot Iteration

```latex
z_{n+1} = z_n^2 + c
```

Use for fractal recursion and self-similarity.

### Mass-Energy Equivalence

```latex
E = mc^2
```

Use as established physics when describing mass-energy equivalence. Metaphysical interpretations must be labeled as interpretation.

### Frequency-Wavelength Relationship

```latex
f = \frac{v}{\lambda}
```

Use for wave frequency discussion.

### Redshift

```latex
z = \frac{\lambda_{\mathrm{observed}} - \lambda_{\mathrm{emitted}}}{\lambda_{\mathrm{emitted}}}
```

Use for cosmological redshift discussion.

### Quantum State Evolution

```latex
i\hbar \frac{\partial}{\partial t}\lvert \psi(t) \rangle = \hat{H}\lvert \psi(t) \rangle
```

Use for quantum state evolution with careful scientific framing.

### Density Matrix

```latex
\rho = \sum_i p_i \lvert \psi_i \rangle \langle \psi_i \rvert
```

Use for decoherence discussion.

### Wave Equation

```latex
\frac{\partial^2 \psi}{\partial t^2} = v^2 \nabla^2 \psi
```

Use for wave-propagation analogy.

### Holographic Entropy Form

```latex
S = \frac{A}{4G\hbar}
```

Review constants, units, and context before publication.

## Symbolic / FHQCM Extensions

### Recursive Time

```latex

```

Used symbolically for recursive time.

### Consciousness Recursion

```latex
C_{n+1} = C_n^2 + c
```

Used symbolically for recursive consciousness, where `c` may represent divine or external input.

### Retrocausal Operator

```latex
\hat{R}\psi(t_2) = \psi(t_1)
```

Used as a symbolic representation of retrocausal influence. Requires careful framing.

### Christic Equivalency

```latex
\text{Wave} = \text{Particle}
\quad
\text{Spirit} = \text{Matter}
\quad
\text{Past} = \text{Future}
```

These are symbolic and theological equivalencies, not literal scientific equations.
