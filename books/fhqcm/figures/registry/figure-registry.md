# FHQCM Figure Registry

This registry makes illustrations a first-class part of the FHQCM book. Every difficult concept should eventually have a plain-language explanation, formal definition, figure, thought experiment, citation/provenance note, and LaTeX reference.

## Status Values

- `draft`: SVG or concept exists but needs refinement.
- `planned`: required for the book but not created yet.
- `prompted`: image-generation prompt exists.
- `generated`: illustration has been generated.
- `review`: needs author/editor review.
- `approved`: ready for manuscript use.
- `print-ready`: high-resolution export exists.

## Required Metadata Per Figure

Each final figure should have:

- figure ID
- title
- chapter/section
- purpose
- reader learning objective
- figure type
- source/provenance
- prompt file
- SVG source file
- PNG or PDF print export
- alt text
- caption
- LaTeX include path
- review status
- approval status

## Rule

No major FHQCM chapter should be considered complete until its difficult concepts have matching figure specs and at least one approved visual aid.
