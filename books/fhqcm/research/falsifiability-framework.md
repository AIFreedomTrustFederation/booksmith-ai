# FHQCM Falsifiability Framework

## Purpose

A theory becomes scientifically useful only when it risks being wrong. This document defines how FHQCM claims should be made falsifiable wherever possible.

## Core Principle

If a claim cannot be tested, measured, modeled, or logically constrained, it must be classified as philosophical, theological, symbolic, or poetic rather than empirical science.

## Claim Classes

| Class | Description | Falsifiability Standard |
| --- | --- | --- |
| Established science | Already supported by mainstream scientific evidence | Cite the literature and do not overextend |
| Mathematical definition | A formal system or symbolic notation | Must be internally consistent |
| Empirical hypothesis | A claim about measurable reality | Must state what observation could disconfirm it |
| Philosophical interpretation | Meaning framework built from evidence and reason | Must state assumptions and alternatives |
| Theological symbol | Faith-based or metaphysical interpretation | Must not be presented as scientific proof |
| Poetic metaphor | Meaning-making language | Must not be confused with mechanism |

## Falsifiability Questions

Every FHQCM hypothesis should answer:

1. What exactly is being claimed?
2. What variables are involved?
3. What would we measure?
4. What result would support the claim?
5. What result would weaken or disprove the claim?
6. What competing explanations exist?
7. What assumptions are required?
8. What part is science, what part is interpretation, and what part is symbol?

## Example Falsification Template

### Hypothesis

Conscious systems exhibit measurable recursive coherence patterns across neural, physiological, and behavioral scales.

### Possible Measurements

- heart-rate variability
- EEG coherence
- respiration patterns
- behavioral timing patterns
- narrative recurrence analysis
- physiological synchrony between people

### Possible Supporting Result

Statistically significant cross-scale coherence appears during meditative, prayerful, emotionally integrated, or high-attunement states compared with baseline states.

### Possible Falsifying Result

No measurable cross-scale coherence difference appears under controlled conditions, or observed coherence is fully explained by known physiological mechanisms without needing FHQCM-specific interpretation.

## Critical Rule

FHQCM must welcome disconfirmation. A model that cannot be corrected cannot mature.
