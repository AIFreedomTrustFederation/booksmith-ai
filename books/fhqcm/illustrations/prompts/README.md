# FHQCM Illustration Prompt Library

This folder stores prompts and production notes for FHQCM illustrations.

## Purpose

The FHQCM book is visually dependent. Its hardest ideas involve recursion, toroidal flow, holographic structure, delayed-choice observation, Christic equivalency, emotional sobriety, and AI stewardship. These should not be left to text alone.

## Prompt File Format

Each prompt file should use this structure:

```markdown
# F001 — FHQCM Overview

Figure ID: F001
Title: FHQCM Overview
Registry Link: ../figures/registry/figure-registry.md
Chapter: Introduction
Type: Conceptual diagram
Audience: General reader
Status: planned

## Purpose

Explain the whole model in one page.

## Visual Requirements

- Consciousness
- Observation
- Quantum Potential
- Fractal Time
- Toroidal Becoming
- Manifest Reality
- Return to Consciousness

## Style

Elegant scientific-spiritual diagram, dark background, gold and blue accents, clean labels, readable in print.

## Avoid

- Overcrowded labels
- Pseudoscientific visual claims
- Illegible tiny text

## Alt Text

A circular flow diagram showing consciousness moving through observation, quantum potential, fractal time, toroidal becoming, manifest reality, and returning to consciousness.

## Caption

The Fractal-Holographic Quantum Consciousness Model as a recursive flow of consciousness, observation, manifestation, and return.
```

## Production Outputs

Each approved figure should eventually produce:

- editable SVG
- print PNG at 300 DPI or better
- optional PDF vector export
- web-optimized PNG/WebP
- alt text
- LaTeX include snippet

## Prompt Principles

- Use figures to clarify, not decorate.
- Keep diagrams readable at book-trim size.
- Separate established science from metaphor.
- Do not imply endorsement from scientists unless properly sourced.
- Include accessibility alt text from the beginning.
