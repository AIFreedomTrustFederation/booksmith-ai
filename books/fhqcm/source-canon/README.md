# FHQCM Source Canon

This folder is the bridge between historical Drive manuscripts, current repo canon, and final book chapters.

## Purpose

The source canon preserves the historical development of the Fractal-Holographic Quantum Consciousness Model while protecting the final manuscript from conceptual drift.

## Included Files

- `drive-eternal-now-extraction.md` — extraction from the Eternal Now / Equivalency Principle PDF stream.
- `drive-fhqcm-main-extraction.md` — extraction from the main FHQCM Drive PDF stream.
- `source-lineage.md` — historical development map.
- `concept-evolution-map.md` — older language mapped to current rigorous terminology.
- `citation-registry.md` — source-to-chapter citation map.

## Rule

Historical claims from Drive may be preserved as canon history, but the manuscript must classify each claim as science, mathematics, philosophy, theology, FHQCM hypothesis, or poetic-symbolic language.
