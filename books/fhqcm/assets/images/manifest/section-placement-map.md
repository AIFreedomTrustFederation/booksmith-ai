# FHQCM Image Section Placement Map

This file maps each image to the section or chapter where it should be introduced.

## Placement Rules

1. Introduce the figure before or immediately after the concept it explains.
2. Do not place full-page art without a reader-learning purpose.
3. Every figure must have a caption, alt text, and provenance note before final publication.
4. Every LaTeX placement should use `latex/figure-includes.tex` macros when possible.
5. If a figure is missing, use a clearly marked placeholder rather than silently omitting the concept.
