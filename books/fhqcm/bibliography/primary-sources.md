# FHQCM Primary Source Registry

This file tracks primary and near-primary sources used to build the FHQCM book.

- Original FHQCM Google Drive white paper
- Booksmith AI repo manuscript files
- Author-approved conversation synthesis
- AI Freedom Trust Publishing metadata
- Canon files under `books/fhqcm/canon/`

## Canonical Source

## Citation Rule

Primary sources may be cited for the development history of the model, but scientific claims must still be supported by external scientific literature.
