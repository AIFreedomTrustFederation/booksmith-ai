# FHQCM Math Registry

This folder stores the formal mathematical and symbolic language of the FHQCM book.

## Files

- `notation.md` — notation rules and style guide
- `symbols.md` — Unicode and LaTeX symbol registry
- `equations.md` — equation registry
- `derivations.md` — derivation workbench
- `glossary.md` — math-specific glossary

## Rule

No equation should enter the final manuscript unless it appears in this registry or in `books/fhqcm/canon/mathematical-definitions.md`.
