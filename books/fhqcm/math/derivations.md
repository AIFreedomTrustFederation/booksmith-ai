# FHQCM Derivation Workbench

## Purpose

This file is where candidate equations are developed before they enter the manuscript.

## Derivation Standard

Each derivation must include:

1. Variables
2. Units or reason units are undefined
3. Assumptions
4. Domain of use
5. Equation
6. Plain-language interpretation
7. Possible proxy measurements
8. Falsification pathway
9. Current status
