# FHQCM Unicode Symbol Registry

## Purpose

This registry defines the beautiful Unicode symbols used in the book and maps them to LaTeX equivalents.

| Symbol | Name | Meaning | LaTeX | Category |
| --- | --- | --- | --- | --- |
| ∞ | infinity | unboundedness, eternity | `\infty` | math/symbolic |
| Ψ | psi | state, wavefunction, or consciousness-state placeholder | `\Psi` | physics/theory |
| ψ | psi lowercase | quantum state or wavefunction | `\psi` | physics |
| Φ | phi | flow field or consciousness-field symbol | `\Phi` | theory/symbolic |
| Ω | omega | completion, maximum coherence, Alpha/Omega symbol | `\Omega` | symbolic/theological |
| α | alpha | beginning, emergence, parameter | `\alpha` | math/symbolic |
| β | beta | transformation parameter | `\beta` | math |
| Σ | sigma | summation or integration of parts | `\Sigma` or `\sum` | math |
| Δ | delta | change or difference | `\Delta` | math |
| ⊕ | circled plus | harmonic union | `\oplus` | symbolic |
| ⊗ | tensor product | recursive interaction or composite state | `\otimes` | math/physics |
| ∮ | closed integral | circulation around a closed path | `\oint` | math/geometry |
| ∴ | therefore | conclusion marker | `\therefore` | logic |
| ∵ | because | reason marker | `\because` | logic |
| ≡ | equivalence | formal equivalence or identity by definition | `\equiv` | math/symbolic |
| ≈ | approximation | similarity, near equality | `\approx` | math |
| → | arrow | transformation or mapping | `\rightarrow` | math |
| ↔ | bidirectional arrow | reciprocal relation | `\leftrightarrow` | math/symbolic |

## Rule

A symbol should not be used decoratively if it is also used formally. Decorative sacred-geometry symbolism must be labeled as symbolic.
