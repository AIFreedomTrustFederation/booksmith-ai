# FHQCM Operator Registry

## Purpose

This registry defines operators used in equations, diagrams, and formal drafts.

| Operator | Name | Meaning | Category |
| --- | --- | --- | --- |
| f(x) | transformation function | maps one state to another | mathematics |
| sum | summation | combines indexed terms | mathematics |
| integral | accumulation across a continuum | mathematics |
| closed integral | circulation around a closed path | mathematics / torus geometry |
| gradient | rate of change across space or field | mathematics |
| tensor product | composite state construction | mathematics / physics |
| arrow | transformation or mapping | notation |
| equivalence | definition-level equivalence or symbolic reconciliation | math / symbolic |

## Rule

Operators must not be used mystically without definition. If an operator is symbolic, it must be labeled symbolic.
