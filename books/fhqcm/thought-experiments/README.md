# FHQCM Thought Experiment Library

This folder stores thought experiments that support hard-to-understand hypotheses and conceptual bridges in the FHQCM manuscript.

Each thought experiment should include:

- purpose
- plain-language setup
- assumptions
- what the reader should notice
- relation to established science
- FHQCM interpretation
- limitations
- matching figure ID
- citation/provenance notes

## Rule

A thought experiment should not be used to overstate a claim. It should help the reader see why a hypothesis is interesting, what it suggests, and what it does not prove.
