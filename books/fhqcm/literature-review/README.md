# FHQCM Literature Review System

This folder organizes source review before the book is built.

## Purpose

FHQCM needs a rigorous literature-review layer that states what each thinker or field actually claims, what is debated, what FHQCM borrows, and what FHQCM extends.

## Review Template

Each source review should include:

- subject or thinker
- primary works
- core claims
- evidence or arguments
- critiques
- what FHQCM borrows
- what FHQCM extends
- what must not be attributed
- citation status
