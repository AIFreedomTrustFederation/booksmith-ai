# FHQCM Publisher Package Checklist

## Front Matter

- title page
- copyright page
- publisher note
- dedication
- epigraph
- foreword
- preface
- acknowledgements
- table of contents

## Scholarly Apparatus

- citation guide
- in-text reference standard
- bibliography
- figure index
- equation index
- glossary
- scientific integrity appendix
- source attribution appendix
- hypothesis registry appendix

## Marketplace Package

- ISBN records
- BISAC categories
- keywords
- short description
- long description
- back-cover copy
- author bios
- publisher bio
- press pitch
- sample chapter

## Production Outputs

- print PDF
- EPUB
- web edition
- cover files
- interior proof
- metadata sheet
- source-provenance packet

## Rule

No public release until author approval, citation review, scientific framing review, and proof review are complete.
