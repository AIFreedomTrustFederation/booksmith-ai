# Booksmith Book Visuals

Booksmith should treat figures and illustrations as first-class publishing assets.

For FHQCM, visual assets should help readers understand recursion, time, torus flow, holographic structure, observation, recovery, and AI stewardship.

## Core Workflow

1. Register the figure.
2. Write the prompt brief.
3. Create SVG or image output.
4. Add alt text and caption.
5. Review for clarity and accuracy.
6. Approve for manuscript use.
7. Export print-ready and web-ready versions.
8. Add the LaTeX reference.

## Required Records

- figure ID
- title
- chapter
- purpose
- audience
- visual type
- prompt
- alt text
- caption
- source file
- print file
- web file
- review status
- approval status
